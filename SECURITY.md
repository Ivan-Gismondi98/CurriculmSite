# Security Policy

## Supported Versions
Attualmente il progetto non ha versioni rilasciate ufficialmente, ma il branch `main` è quello attivo e mantenuto.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability
Se trovi una vulnerabilità nel progetto, ti invito a segnalarla in modo responsabile:

- **Come segnalare**: Apri una issue con il tag `security` oppure invia una mail a [tuo indirizzo email].
- **Cosa includere**:
  - Descrizione dettagliata del problema
  - Passaggi per riprodurlo
  - Eventuali suggerimenti per la risoluzione
- **Tempi di risposta**: Cercherò di rispondere entro 7 giorni.
- **Cosa aspettarsi**:
  - Se la vulnerabilità è confermata, verrà corretta il prima possibile.
  - Se non è considerata critica, verrà pianificata per una release futura.
